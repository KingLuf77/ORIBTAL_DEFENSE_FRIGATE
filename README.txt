ORBITAL DEFENSE FRIGATE
Made by: KingLuf
Completed: 10-30-2023

Orbital Defense Frigate is a text-based retro sci-fi space shooter.

This game uses libaries only available in Visual Studio.

Install guide:
1) Download ORBITAL_DEFENSE_FRIGATE.zip
2) Open ORBITAL_DEFENSE_FRIGATE.sln
3) Run the game

Tips:
- Hit F11 to run the game in fullscreen
- Hit ctrl+ to zoom in the game

If you want music:
4) Set "int music = 0" to "int music = 1"
5) Run the game
6) When the game starts, VLC or Windows Media Player will open the song file
