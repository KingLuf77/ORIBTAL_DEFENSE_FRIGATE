ORBITAL DEFENSE FRIGATE
Made by: KingLuf
Completed: 10-30-2023

Orbital Defense Frigate is a text-based retro sci-fi space shooter.

This game uses libaries only available in Microsoft Visual Studio.

Install guide:
1) Download all files and place them in a folder.
2) Open ORBITAL_DEFENSE_FRIGATE.sln
3) Run the game

Tips:
- Hit F11 to make the game fullscreen
- Hit CTRL+ to zoom in the game

If you want music:
4) Change "int music = 0" to "int music = 1".
5) Run the game.
6) When the game starts, VLC or Windows Media Player will open the song file. Hit Alt-Tab to return to the game screen.
